Niveaux piézométriques mesurés sur les forages BSS dans un rayon de 10 km — données Hub'Eau API niveaux_nappes. Ces valeurs servent à vérifier que la crépine de chaque piézomètre sera positionnée dans la zone saturée de l'aquifère.

NOTE RÉGLEMENTAIRE

Le Code Minier L.411-1 (BSS — Banque du Sous-Sol) s'applique à tout ouvrage de reconnaissance ou d'exploitation du sous-sol dépassant 10 mètres de profondeur. Une déclaration obligatoire doit être effectuée auprès de la DREAL et le forage doit être enregistré dans la Base de Données du Sous-Sol (h2geo.brgm.fr).
