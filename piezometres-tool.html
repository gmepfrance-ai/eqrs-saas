Recommandations d'implantation

Test IOTA R.214-1 — Code de l'Environnement

Loi sur l'Eau : Les sondages, forages, créations de puits ou d'ouvrages souterrains sont soumis à la nomenclature des Installations, Ouvrages, Travaux et Activités (IOTA) selon l'article R.214-1 du Code de l'Environnement.

Rubriques applicables R.214-1

📋 Code Minier L.411-1 — Ouvrages profonds

Les forages de profondeur ≥ 10 m sont également soumis à déclaration auprès du BSS (Banque du Sous-Sol, BRGM) selon l'article L.411-1 du Code Minier. L'enregistrement BSS est obligatoire dans un délai de 2 mois après réalisation.
